    <h1>Movie Recommendation System</h1>
    <p>This project is a <strong>Movie Recommendation System</strong> that utilizes data preprocessing and machine learning techniques to provide personalized movie suggestions based on user preferences.</p>
    <h2>Features</h2>
    <ul>
        <li>Cleans and preprocesses movie dataset to remove inconsistencies.</li>
        <li>Handles missing values, outliers, and categorical variable encoding.</li>
        <li>Implements feature selection and scaling for optimized performance.</li>
        <li>Uses machine learning models to recommend movies based on user input.</li>
        <li>Supports large-scale movie datasets for better recommendations.</li>
    </ul>
    <h2>Installation</h2>
    <ol>
        <li>Clone this repository:<br>
            <code>git clone https://github.com/yourusername/movie-recommendation.git</code>
        </li>
        <li>Navigate to the project directory:<br>
            <code>cd movie-recommendation</code>
        </li>
        <li>Install required dependencies:<br>
            <code>pip install -r requirements.txt</code>
        </li>
        <li>Ensure you have the dataset (<code>movie-dataset.csv</code>) in the project directory.</li>
    </ol>
    <h2>Usage</h2>
    <ol>
        <li>Run the Jupyter Notebook:<br>
            <code>jupyter notebook</code>
        </li>
        <li>Open <code>movie.ipynb</code> and execute the cells step by step.</li>
        <li>Follow the prompts to get movie recommendations.</li>
    </ol>
    <h2>Technologies Used</h2>
    <ul>
        <li><strong>Python</strong></li>
        <li><strong>Pandas</strong> for data manipulation</li>
        <li><strong>NumPy</strong> for numerical computations</li>
        <li><strong>Scikit-learn</strong> for machine learning preprocessing</li>
        <li><strong>Jupyter Notebook</strong> for interactive execution</li>
    </ul>
